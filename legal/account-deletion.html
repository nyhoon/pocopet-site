계정 및 관련 데이터 삭제 안내

시행일 2026-05-15 · Google Play 「계정 삭제 URL」용 · 로그인 없이 열람 가능

요약
  • 계정 삭제는 앱 안에서 직접 할 수 있습니다. (가장 빠른 방법)
  • 앱을 사용할 수 없는 경우 아래 이메일로 삭제를 요청할 수 있습니다.
  • 탈퇴 시 이 기기의 펫·대화·개인화 기억 등과, 연결된 서버(Supabase) 계정·프로필이 삭제됩니다.

1. 앱에서 직접 계정 삭제하기 (권장)

아래 순서대로 진행하면 계정 및 관련 데이터 삭제를 요청할 수 있습니다.

삭제 절차
  1. PocoPet 앱을 실행하고 로그인합니다.
  2. 하단 또는 메뉴에서 「설정」으로 이동합니다.
  3. 「프로필 및 계정」을 선택합니다.
  4. 「회원 탈퇴」를 탭합니다.
  5. 이메일로 가입한 경우: 비밀번호를 입력한 뒤 안내 화면에서 「탈퇴하기」를 눌러 최종 확인합니다.
  6. 카카오·네이버·구글로 가입한 경우: 안내에 따라 해당 SNS로 한 번 더 인증(재로그인)한 뒤 탈퇴를 완료합니다.

탈퇴 확인 화면에서 안내하는 바와 같이, 이 기기에 저장된 펫 정보·뽑기 포인트·대화 기록·개인화 기억·설정 등이 삭제되며, 원격 계정을 사용 중이었다면 서버의 계정·프로필·연동 데이터도 함께 삭제 처리됩니다. 삭제 후에는 되돌릴 수 없습니다.

2. 앱을 사용할 수 없을 때 (이메일 요청)

기기 분실·로그인 불가 등으로 앱에서 탈퇴할 수 없는 경우, 아래로 연락해 주세요.

  • 이메일: feelsogxxd@naver.com
  • 제목 예: PocoPet 계정 삭제 요청
  • 본문에 포함할 정보: 가입에 사용한 이메일 주소 또는 SNS 종류(카카오/네이버/구글), 삭제 요청 사실

본인 확인을 위해 추가 정보를 요청할 수 있으며, 확인 후 영업일 기준 약 14일 이내 처리를 목표로 합니다.

3. 삭제되는 데이터 유형

계정 탈퇴가 완료되면 다음 데이터가 삭제되거나 복구 불가 상태가 됩니다.

데이터 유형 저장 위치 삭제 시 처리
계정·로그인 정보 (이메일, 인증 토큰, SNS 연동) 기기 + Supabase Auth 삭제
프로필 (닉네임 등) 서버 + 기기 삭제
펫 정보·스탯·뽑기 포인트·미션·채팅 보상 상태 기기 + 서버(동기화 시) 삭제
대화 기록·상호작용 로그 기기(Room DB) 삭제 (탈퇴 시 로컬 정리)
개인화 기억(사실·요약) 기기 삭제
앱 설정(펫 크기, 권한 안내 상태 등) 기기 삭제
인앱 결제·패스·재화 원장 연동 서버(ledger 등) 계정에 묶인 데이터 삭제 또는 비식별 처리

4. 일부 보관·지연 삭제되는 데이터

다음은 법령·분쟁 대응·서비스 안정성을 위해 즉시 삭제되지 않거나 제한적으로 보관될 수 있습니다.

데이터 유형 보관 사유 추가 보관 기간 (목표)
서버 백업·복제본 장애 복구 백업 주기에 따라 최대 약 30일 이내 순차 삭제
결제·환불·세무 관련 기록 전자상거래 등 관련 법령 관련 법령이 정한 기간 (통상 5년 이내)
보안·부정 이용 방지 로그 서비스 보호 목적 달성 시까지, 통상 90일 이내
Google AdMob 등 광고 SDK 데이터 광고 사업자 정책 Google 정책에 따름 (앱 계정 삭제와 별도)
Google Gemini로 전송된 대화 텍스트 클라우드 AI 처리 Google API 정책·설정에 따름. 앱은 탈퇴 후 로컬·서버 보관 데이터를 삭제합니다.

5. 계정 없이 삭제할 수 있는 데이터

로그인 상태에서 다음만 삭제할 수도 있습니다. (계정 전체 탈퇴와는 별개입니다.)

  • 설정 → 기억(개인화) → 전체 삭제: 온디바이스 개인화 기억(사실·요약)만 삭제
  • 앱 삭제(제거): 기기에 남은 로컬 DB·설정은 OS에 따라 제거됩니다. 서버 계정은 자동으로 삭제되지 않으므로 서버에 가입한 경우 반드시 회원 탈퇴 또는 이메일 요청을 해 주세요.
